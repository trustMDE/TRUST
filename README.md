# Welcome to the TRUST Companion Material

This repository contains companion material related to implementations of TRUST instantiations.

## Repository Contents

### `./SysMLv2`

The `./SysMLv2` directory contains:

- a GPT-5.5-generated test PDF specification for a fictitious embedded cinema amplifier with a Dolby 6+1-compatible architecture;
- a set of SysMLv2 models produced from that specification by TRUST's SysMLv2 instantiation.
